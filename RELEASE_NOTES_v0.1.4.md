# SoE Holy Grail v0.1.4

Manual Mode update.

## Changes

- Added a Manual Mode setting that pauses automatic save-change scans and interval scans.
- Manual Stash Sync and Player Sync buttons still work while Manual Mode is enabled.
- Automatic scans after selecting a stash file, save folder, or active character now respect Manual Mode.
- Player sync settings now clearly indicate when Manual Mode is pausing automatic player sync.

## Install

Download `SoE-Holy-Grail-win32-x64-v0.1.4.zip`, extract it anywhere, and run `SoE Holy Grail.exe`.

Players on older versions can use Settings -> Check Updates after this release is published.
