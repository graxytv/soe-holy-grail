# GitHub Description

## Repository About

Sanctuary of Exile holy grail tracker with automatic stash and character scanning, Fate Card stack completion, overlay progress, sounds, and in-app updates.

## Release Description

SoE Holy Grail is a Windows desktop tracker for Sanctuary of Exile players who want automatic grail progress without needing to keep every item forever.

It tracks unique items, set items, runes, and Fate Cards. The app scans your shared stash and selected character save, marks new grail completions, and keeps them completed permanently after first discovery. Fate Cards only complete when the required full stack has been found once.

### Highlights

- Automatic shared stash and character save scanning.
- Permanent grail completion history.
- Unique, set, rune, and Fate Card tracking.
- Fate Card full-stack completion logic.
- Character picker with Diablo II class portraits.
- Recent Finds list.
- Optional progress circle overlay.
- Configurable new-grail sounds.
- In-app update checking through GitHub Releases.
