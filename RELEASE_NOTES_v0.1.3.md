# SoE Holy Grail v0.1.3

Unique identity hotfix.

## Changes

- Fixed unique amulets and rings that could resolve as the next item with the same base code, such as Nokozan Relic appearing as The Eye of Etlich.
- Corrected SoE expansion unique save IDs so items like Saracen's Chance, Goldrim, and Seven-League Step still scan correctly.
- Restored exact unique/set ID matching before fallback matching.
- Repaired the generated catalog without changing existing manual grail progress behavior.

## Install

Download `SoE-Holy-Grail-win32-x64-v0.1.3.zip`, extract it anywhere, and run `SoE Holy Grail.exe`.

Players on older versions can use Settings -> Check Updates after this release is published.
