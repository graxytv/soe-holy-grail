# SoE Holy Grail v0.1.1

Small quality-of-life update for grail management.

## Changes

- Added a guarded Reset Grail button in Settings.
- Reset clears found items and recent finds while keeping stash path, active character, overlay, sound, and update settings unchanged.
- The reset action asks for confirmation before clearing progress.

## Install

Download `SoE-Holy-Grail-win32-x64-v0.1.1.zip`, extract it anywhere, and run `SoE Holy Grail.exe`.

Players on v0.1.0 can use Settings -> Check Updates after this release is published.
