# SoE Holy Grail v0.1.0

First public release of SoE Holy Grail by Graxy_TV.

## Highlights

- Sanctuary of Exile-only grail tracker.
- Tracks unique items, set items, runes, and Fate Cards.
- Automatically scans the selected shared stash and active character save.
- Saves grail completions permanently after first discovery.
- Fate Cards complete once a full required stack is found.
- Character picker with Diablo II class portraits.
- Optional progress circle overlay with new-find popup.
- Recent Finds panel for the latest 20 grail completions.
- Configurable new-grail sound with SoE Companion and FilterBlade options.
- In-app update checking and portable runtime update install support.
- Correctly resolves adjacent same-code uniques and sets, including amulets/rings like Saracen's Chance vs Highlord's Wrath.

## Install

Download `SoE-Holy-Grail-win32-x64-v0.1.0.zip`, extract it anywhere, and run `SoE Holy Grail.exe`.

## Update Notes

This release is the updater baseline. Future releases should upload a Windows portable zip named like:

```txt
SoE-Holy-Grail-win32-x64-v0.1.1.zip
```

The in-app updater checks releases at `graxytv/soe-holy-grail`.
