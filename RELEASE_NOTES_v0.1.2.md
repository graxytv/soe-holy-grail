# SoE Holy Grail v0.1.2

Player sync testing update.

## Changes

- Split manual syncing into Stash Sync and Player Sync buttons.
- Player Sync reads only the selected active character `.d2s` file.
- Added optional timed active-player sync in Settings.
- Added a slider to control the player sync interval from 3 to 60 seconds.
- Stash syncing now stays stash-focused, while player syncing is opt-in for interval testing.

## Install

Download `SoE-Holy-Grail-win32-x64-v0.1.2.zip`, extract it anywhere, and run `SoE Holy Grail.exe`.

Players on older versions can use Settings -> Check Updates after this release is published.
